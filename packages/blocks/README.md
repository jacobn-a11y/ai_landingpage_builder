# @replica-pages/blocks

Shared block and form type definitions for the Replica Pages editor.

## Usage

```ts
import type { PageContentJson, FormSchemaJson } from '@replica-pages/blocks';
```

## Exports

- **block-types.ts** - Block categories, types, `PageContentJson`, `BaseBlock`
- **form-types.ts** - Form field types, `FormSchemaJson`, `PageFormBindingType`, `FieldMapping`

## Build

```bash
npm run build
```

Output is written to `packages/blocks/dist`.
